# Examples

This directory contains example code for integrating with the Chivox Speech Evaluation Function Calling API.

## Examples Overview

| Directory | Description | Language |
|-----------|-------------|----------|
| [proxy-server/](proxy-server/) | Secure API Key proxy server with web UI | Python (Flask) |
| [quick-test/](quick-test/) | Minimal test scripts | Python, Node.js, curl |

## Which example should I use?

- **Just want to try it?** Start with [quick-test/](quick-test/) — simple curl/Python/Node.js scripts
- **Building a web app?** Use [proxy-server/](proxy-server/) as a starting point
- **Testing the API programmatically?** Check [quick-test/](quick-test/)
